My project


Attendance Monitoring using
Arduino and RFID


We did this project using arduino and rfid when
I was in my first year at my old school. Its function
is that when the person registered in the system enters
an rfid, it will save the time and day that they entered.
And when the person entering the system uses an unregistered
rfid, the buzzer will sound and the red light will turn on.
